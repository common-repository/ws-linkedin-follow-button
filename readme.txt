== Description ==

WS LinkedIn Follow Button plugin provides a small button displayed on your websites to help users easily follow a Company/Showcase Page. 

<strong>Features</strong> 

* Easy to install and Configure
* User friendly and easy to use.
* Simply Add Follow Button
* Change language of follow button
* Easy to add shortcode anywhere in your wordpress site.

== Installation ==

1. Download [WS LinkedIn Follow Button](https://wordpress.org/plugins/ws-linkedin-follow-button/) plugin.   
2. Upload the WS LinkedIn Follow Button plugin from your admin panel.    
3. Activate the WS LinkedIn Follow Button plugin. 
4. Add WS LinkedIn Follow Button widget to your sidebar.
